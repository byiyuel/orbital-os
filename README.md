### Visualize the Global Grid. Decrypt Economic History.

Raw economic data is often trapped in sterile spreadsheets or static charts that fail to show the spatial and temporal connectivity of our world. **Orbital OS** breaks these barriers, transforming dry World Bank metrics into a high-fidelity, interactive 3D tactical interface designed for the modern explorer.

---

## 🚀 Experience the Advantage

Most data portals feel like textbooks; Orbital OS feels like a command center.

- **Interactive 3D Chronology**
  Stop looking at snapshots. Use the "Time Machine" slider to watch wealth migrate and economies fluctuate across decades in a seamless 3D space.
- **Tactical Benchmarking Interface**
  Leverage a built-in terminal to "scan" country nodes and initiate side-by-side comparative analysis with zero-latency visual feedback.
- **Low-Frequency Aesthetic, High-Frequency Data**
  A custom-engineered "OS" HUD that makes complex macro-economic indicators—GDP, Inflation, and Growth—actually engaging to explore.

---

## 🛠 Tech Stack

Built with a focus on performance, type safety, and immersive visuals:

- **Framework:** [Next.js 15](https://next.js) (App Router)
- **Visuals:** [Three.js](https://threejs.org) & [D3.js](https://d3js.org) for hybrid 3D/Geospatial rendering
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com) & [Framer Motion](https://www.framer.com/motion/)
- **Data Source:** [World Bank API](https://data.worldbank.org/) (Real-time UPLINK)

## 🧩 Terminal Commands

Once the system is active, press `Ctrl + \`` to toggle the Command Shell:

- `scan [ISO_CODE]` — Navigate to a specific country node (e.g., `scan TUR`).
- `compare [C1] [C2]` — Benchmarks two countries side-by-side.
- `top [gdp|inflation]` — Displays the current top 10 global nodes by indicator.
